Number Bond Cups

Addition 6-8 years Game Free Download

About This Activity

Number Bond Cups is a fun, hands-on game that helps children understand number bonds and addition strategies. By physically manipulating cups with numbers, children can visualize how numbers can be broken down and recombined in different ways.

Learning Objectives

  1. Understand number bonds to 10
  2. Develop addition fluency
  3. Recognize patterns in number relationships
  4. Build problem-solving skills

Materials Needed

  1. 10 paper or plastic cups
  2. Marker or printable number labels (included in download)
  3. Small objects for counting (beans, buttons, etc.)
  4. Activity worksheet (included in download)

Setup Instructions

  1. Label each cup with numbers 1 through 10
  2. Place the corresponding number of counting objects in each cup
  3. Print the activity worksheet
  4. Clear a flat surface for playing

How to Play

The basic game involves selecting a "target number" and then finding different combinations of cups (number bonds) that add up to that target. For example, if the target is 10, children might select cups labeled 7 and 3, or 5 and 5, or 1, 4, and 5.

The activity includes several variations and challenges for different skill levels, all detailed in the downloadable instructions.

Extension Ideas

  1. Use the cups for subtraction practice
  2. Create number bond challenges for larger numbers
  3. Time challenges to build fluency
  4. Create word problems using the cup numbers
